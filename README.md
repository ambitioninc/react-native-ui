# react-native-ui

`react-native-ui` is a collection of components for [React Native](http://facebook.github.io/react-native/).

## Install

To install the latest release:
```shell
npm install react-native-ui
```

To build the latest code directly from source:
```shell
npm install git://github.com/ambitioninc/react-native-ui.git
```
